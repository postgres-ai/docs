Database Lab Platform uses the following service providers:

- [Google Cloud Platform (GCP)](https://cloud.google.com/)
- [Amazon Web Services (AWS)](https://aws.amazon.com/)
- [Okmeter](https://okmeter.io/)
- [Google Analytics (GA)](https://analytics.google.com/analytics/web/)
- [Mailgun](https://www.mailgun.com/)
- [Intercom](https://www.intercom.com/)
- [Hubspot](https://www.hubspot.com/)
- [LinkedIn](https://linkedin.com/)
- [Typeform](https://www.typeform.com/)
- [GitLab](https://gitlab.com)
- [Stripe](https://stripe.com)
- [Slack](https://slack.com)
- [Airtable](https://airtable.com/)

This list was defined in February 2021 and it is well-maintained (should be there are any changes, the list will be edited).

Users' personal information can be shared with some of the Platform's service providers. These are companies who provide services on Postgres.ai's behalf, such as hosting our Services, marketing, advertising, social, analytics, support ticketing, credit card processing, security and other such similar services. These companies are subject to contractual requirements that govern the security and confidentiality of users information.

See also: [Privacy Policy](https://postgres.ai/privacy).
