## GUI
1. With Joe chatbot sessions (based Database Lab clones) you can perform any changes and not be afraid to damage the data. For example, connect to your clone and drop any table.

<!--DOCUSAURUS_CODE_TABS-->
<!--Web UI-->
![Joe bot / Change data / Web UI](/docs/assets/guides/reset-clone-web-1.png)
<!--Slack-->
![Joe bot / Change data / Slack](/docs/assets/guides/reset-clone-slack-1.png)
<!--END_DOCUSAURUS_CODE_TABS-->

2. Execute [reset](/docs/joe-bot/commands-reference#reset) command.

<!--DOCUSAURUS_CODE_TABS-->
<!--Web UI-->
![Joe bot / Reset the state / Web UI](/docs/assets/guides/reset-clone-web-2.png)
<!--Slack-->
![Joe bot / Reset the state / Slack](/docs/assets/guides/reset-clone-slack-2.png)
<!--END_DOCUSAURUS_CODE_TABS-->

3. Wait for ✅ **OK** status. The data will be recovered to the initial state.

<!--DOCUSAURUS_CODE_TABS-->
<!--Web UI-->
![Joe bot / Status OK / Web UI](/docs/assets/guides/reset-clone-web-3.png)
<!--Slack-->
![Joe bot / Status OK / Slack](/docs/assets/guides/reset-clone-slack-3.png)
<!--END_DOCUSAURUS_CODE_TABS-->
