---
title: Install and initialize Database Lab CLI
---

- Run `curl https://gitlab.com/postgres-ai/database-lab/-/raw/master/scripts/cli_install.sh | bash` to install Database Lab CLI.
- Run `dblab init --environment-id=demo --url=https://URL --token=TOKEN`. Use URL of your instance and personal or verification token.
- Test your configuration with instance status request `dblab instance status`.
