# PostgresAI rules

Development and writing standards for PostgreAI projects.

## Structure

Rules are organized into chapters:
- **Writing** (`writing__*`) — Communication and documentation standards
- **Development** (`development__*`) — SQL, database design, and PostgreSQL development

## Usage

**For AI systems**: Import as Git submodule
```bash
# Latest version
git submodule add https://gitlab.com/postgres-ai/rules.git .cursor

# Specific version
git submodule add -b v0.2 https://gitlab.com/postgres-ai/rules.git .cursor
```

**For humans**: Read and follow the guidelines
